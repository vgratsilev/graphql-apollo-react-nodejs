## GraphQL & Apollo & React & Nodejs

### How to start
_ You can open https://vgratsilev.github.io/graphql-apollo-react-nodejs/
- Or run apps locally:
  - In folder `server` execute `npm run dev`
  - In the folder `client` execute `npm start`
